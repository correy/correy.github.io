  • Slater Systems, LLC April 2018 - June 2019
    Slater is a new kind of software for all devices, built on event-sourcing over end-to-end encrypted messaging. It's gradually-specified, flow-based logic in secure messaging threads (with or without other humans chatting in them). It enables a "Wizard of Oz" approach where you can pretend to be the software you want in order to capture user story data before really implementing it. It integrates all kinds of graphical widgets as message types, with slash commands, and later, optional conversational AI running in situ. And then it tiles these message threads over as many devices as you can throw at it, homogenizing and integrating devices.
    I've failed to balance this pursuit with paying work, and I've made a mess of the project and my life because of it... but I still think it was worth it, to explore the space that I see. And my guts concur. Software ate the world, but now Slater wants to eat all that software.
    used: JavaScript, NodeJS, FaunaDB, Choo, Tachyons, Sodium, Wire Open Source, forrays into the hypercore (DAT) ecosystem for evaluation / discussion
